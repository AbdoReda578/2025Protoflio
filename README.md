## ğŸ“ Customization

### Update Personal Information

Edit the content in each component file:
- `Hero.tsx` - Name and tagline
- `About.tsx` - Bio and statistics
- `Skills.tsx` - Technical skills
- `Projects.tsx` - Project details
- `Education.tsx` - Education and awards
- `Contact.tsx` - Contact information

### Modify Theme

Update colors and styles in `src/index.css`:
- CSS variables for colors
- Custom utility classes
- Animation keyframes

### Add CV

Replace `public/cv.pdf` with your actual CV file.
